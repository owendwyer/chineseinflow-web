# Chinese In Flow — Game

CreateJS canvas game for practicing Chinese characters.

See the [root README](../../README.md) for build and deploy instructions.

## Local development

Open `Main.html` via a local server (e.g. from the repo root after `npm run build`, use `npm run dev`).

## Build

From the repo root:

```bash
npm run build
```

Or from this directory:

```bash
npm run dist
```

Output goes to `dist/` here, then the root build copies it to the site `dist/js/`.
